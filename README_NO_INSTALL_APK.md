# NexusAlchemy Player — офлайн APK без Android Studio

Цей репозиторій вже містить HTML/JS застосунок у `app/src/main/assets/`.
APK збирається **в хмарі** через GitHub Actions — тобі не треба нічого встановлювати.

## 1) Як зібрати APK (тільки браузер)
1. Створи новий репозиторій на GitHub.
2. Завантаж сюди ВСІ файли з цього архіву (кнопка **Add file → Upload files**).
3. Перейди у вкладку **Actions**.
4. Відкрий workflow **Build APK** і натисни **Run workflow**.
5. Після завершення збірки зайди в run → **Artifacts** → скачай `nexus-alchemy-debug-apk`.

> Так, APK доведеться скачати — інакше його неможливо встановити на телефон.

## 2) Де лежить APK
GitHub Actions віддає артефакт:
`app-debug.apk`

## 3) Як замінити веб-частину (якщо захочеш)
Поклади свої файли у:
`app/src/main/assets/`
(і знову запусти workflow)

## 4) Встановлення на Android
На телефоні відкрий APK і дозволь встановлення з невідомих джерел (як попросить система).
