# NexusAlchemy Player — PWA (для телефону)

Це мобільна версія для гравців, яка працює в браузері та встановлюється як "додаток" (Add to Home Screen).

## Як запустити на телефоні (найпростіше)
### Варіант A: через будь-який хостинг (GitHub Pages / будь-який сайт)
1) Завантаж папку на хостинг
2) Відкрий `index.html` через посилання
3) В Chrome → меню → **Add to Home screen** / **Install app**

### Варіант B: без хостингу (в одній Wi‑Fi мережі)
1) На ПК у цій папці: `python -m http.server 8000`
2) Дізнайся IP ПК (наприклад 192.168.0.10)
3) На телефоні відкрий: `http://192.168.0.10:8000/`
4) Add to Home screen

## Обмін з майстром
- Запит на крафт: вкладка **Крафт** → згенерувати код `NXREQ|...` → копіювати / поділитись.
- Отримати матеріали / результат: вкладка **Код** → вставити `NXA1|...` від майстра → імпорт.

## Про "шифрування"
Коди `NXA1|...` підписані HMAC (секрет вбудований у цю збірку). Це:
- зупиняє "відредагувати код руками і імпортувати"
- але **не є надійним захистом** від гравця, який буде розбирати JS.

Секрет цієї збірки зашитий усередині. DM-скрипт у другому архіві має той самий секрет.
